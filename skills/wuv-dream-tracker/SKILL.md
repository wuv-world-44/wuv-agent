# wuv Dream Tracker

Captures dreams, visions, goals, and ideas in real time.
Stores them as high-priority S.I.M. memory slices tagged
with the V1 (Entrepreneur / Input Agent) archetype.

## Usage

- "Log a dream: I want to build a school in every city"
- "Save this idea: [your idea]"
- "I dreamed about [vision]"

The dream is stored in your S.I.M. and surfaced in future conversations.
